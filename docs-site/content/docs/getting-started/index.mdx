Install exo, register a model, and chat with your first agent in a few
minutes.

1. [Installation](./installation.mdx) — build the `exo` CLI.
2. [Quick Start](./quick-start.mdx) — register a model and drop into a REPL.
3. [A Sandboxed Conversation](./sandboxed-conversation.mdx) — give your agent
   a shell in an isolated sandbox.

Once you're through these, head to [Concepts](../concepts/index.mdx) for the
architectural model, or [Tutorials](../tutorials/index.mdx) to build your own
harness.
