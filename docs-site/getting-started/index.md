# Getting Started

Install exo, register a model, and chat with your first agent in a few
minutes.

1. [Installation]({% link getting-started/installation.md %}) — build the
   `exo` CLI.
2. [Quick Start]({% link getting-started/quick-start.md %}) — register a
   model and drop into a REPL.
3. [A Sandboxed Conversation]({% link getting-started/sandboxed-conversation.md %})
   — give your agent a shell in an isolated sandbox.

Once you're through these, head to
[Concepts]({% link concepts/index.md %}) for the architectural model, or
[Tutorials]({% link tutorials/index.md %}) to build your own harness.
