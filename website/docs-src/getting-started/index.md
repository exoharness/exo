# Getting Started

Get a fully functional agent — sandbox, scheduler, browser chat — with one
script, then learn what to do with it.

1. [Installation](./installation) — run `setup.sh`; it handles keys,
   build, and launch.
2. [Your First Session](./first-session) — test prompts, prompt
   customization, and the `./exo.sh` control surface.

Building from primitives instead of using the canonical agent:

3. [Using the CLI Directly](./quick-start) — secrets, model bindings,
   and a bare REPL.
4. [A Sandboxed Conversation](./sandboxed-conversation) — give a bare
   agent a shell in an isolated sandbox.

Once you're through these, head to [Concepts](../concepts/index) for the
architectural model, or [Tutorials](../tutorials/index) to build your
own harness.
